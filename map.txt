================================================================================
PROCESSING FLOW (CURRENT DEFAULT PATH)
================================================================================

1. Startup + path validation in main() (parse args, validate source/csv/shapefiles, mkdir output/temp).
2. Instantiate ChartSlicer and apply runtime options/compatibility warnings from CLI flags.
3. If --postprocess-only: call _run_postprocess_upload_stage() and return (no review/warp/mosaic).
4. Full path: load_dole_data(), then _build_source_index() and _build_shapefile_index().
5. Preview/review phase: generate_review_report() writes review CSV and prompts for confirmation.
6. Prompt for all dates or a YYYY-MM-DD date range in main(), then call process_all_dates(date_range).
7. process_all_dates(): filter date keys, download missing sources up front via _download_missing_files().
8. Resume support: rebuild per-location/date runtime library via _rebuild_vrt_library().
9. Per date: group CSV rows by (location, edition), find sources/shapefiles, build warp jobs.
10. Warp phase: adaptive ThreadPoolExecutor runs _warp_worker() -> warp_and_cut(), with duplicate-row fallback.
11. Collect valid warped rasters into vrt_library and build a date mosaic VRT via build_vrt().
12. Queue (mosaic_vrt, date) into postprocess_jobs for downstream conversion/upload.
13. Postprocess phase: _run_postprocess_upload_stage() dedupes jobs, tunes workers, runs postprocess_tif_worker().
14. Worker converts raster -> tiles -> MBTiles -> PMTiles and uploads via rclone, with status heartbeat JSON files.

================================================================================
KEY RUNTIME STATE (CURRENT)
================================================================================

- self.dole_data: Dict[date_key, List[csv_row]] populated by load_dole_data()
- self.date_key_map: Dict[date_key, (start_date, end_date)] for range sorting/filtering
- self.files_by_name + self.tifs_by_dir: source lookup caches built by _build_source_index()
- self.shapefile_index: normalized location -> shapefile path cache
- vrt_library (runtime local in process_all_dates): per-location/date warped raster registry
- postprocess_jobs (runtime local in process_all_dates): queue of (mosaic_vrt, date) tuples
- jobs_by_stem (runtime local in _run_postprocess_upload_stage): dedupe latest input per output stem
- self.upload_root / self.postprocess_tiles_temp_root: local PMTiles output and temp tile/status directories
